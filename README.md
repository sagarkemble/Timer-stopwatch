# Stopwatch & Timer

A simple and elegant stopwatch and timer application built with React, TypeScript, Vite, and **shadcn/ui** components.

# Stopwatch & Timer

## Tech Stack

- **React** - UI framework
- **TypeScript** - Type-safe development
- **Vite** - Fast build tool
- **shadcn/ui** - High-quality component library
- **Tailwind CSS** - Utility-first styling

## Getting Started

```bash
# Install dependencies
bun install

# Run development server
bun dev

# Build for production
bun run build
```

## Project Structure

- `/src/components` - React components including timer, stopwatch, and shadcn/ui components
- `/src/hooks` - Custom React hooks for timer and stopwatch logic
- `/src/assets` - Static assets and animations
